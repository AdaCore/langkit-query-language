# LKQL language support extension

(see https://docs.adacore.com/live/wave/lkql/html/gnatcheck_rm/gnatcheck_rm/lkql_language_reference.html# for more info)

## Features

* Syntax highlighting

## Planned features

* Semantic highlighting
* Code navigation
* Variable renaming
* Completion support
